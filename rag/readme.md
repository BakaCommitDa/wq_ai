# RAG


- Function Call 给llm 与外部资源的通信协议
- MCP 定义了LLM 与外部资源的通信协议

RAG（检索增强生成）是一种通过从外部知识库检索相关信息并将其融入提示词，来增强大语言模型生成内容的准确性、
时效性和相关性的技术框架。

RAG = 让大模型先“翻资料”再回答，答得更准、不瞎编。
